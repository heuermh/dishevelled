Simple observable decorators for collection and map interfaces.

Summary

The decorators in this package simply fire empty vetoable change events in preXxx methods and empty change events in postXxx methods. Listeners may query the source of the events to determine what may or may not have changed due to the event.

Example

For example, consider a listener interested in changes in size of an observable collection of strings: 

class SizeChangeListener<String>
  implements CollectionChangeListener<String>, VetoableCollectionChangeListener<String>
{
  private int sizeBefore;
  private int sizeAfter;

  public void collectionWillChange(final VetoableCollectionChangeEvent<String> event)
    throws CollectionChangeVetoException
  {
    ObservableCollection<String> source = event.getObservableCollection();
    sizeBefore = source.size();
  }

  public void collectionChanged(final CollectionChangeEvent<String> event)
  {
    ObservableCollection<String> source = event.getObservableCollection();
    sizeAfter = source.size();
    if (sizeAfter != sizeBefore)
    {
      // collection size changed, do something...
    }
  }
}
